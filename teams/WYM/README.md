## Idea Chosen
Smart Exam Preparation Planner with an interactive node-based learning roadmap.

## Problem Statement
Students often struggle with overloaded syllabi, missed study sessions, and last-minute cramming. Fixed timetables do not adapt well when a day is missed, which can quickly create stress, poor prioritization, and burnout. Aegis addresses this by organizing study work into a flexible, priority-based system that helps students stay on track even when plans change.

## Implementation Details
Aegis is built as a multi-page React dashboard with a shared layout shell and separate views for the main dashboard, calendar, syllabus map, analytics, and creation flows. The UI focuses on study tracking, daily focus, and visual planning rather than static lists.

The dashboard highlights core metrics such as burnout risk, performance, and daily focus. The calendar view manages study sessions and the timeline view helps users see what comes next. The syllabus section uses a node-based structure to represent learning topics and dependencies, making the roadmap easier to understand and update.

State management is handled with Zustand, and the app is structured so it can later support API-driven scheduling, progress tracking, and recalculation when sessions are missed.
